To run the script (requires boto3):

python3 ./create_dashboard_app.py <profile> <prod-repo-stack-id> <prod-workers-stack-id> <prod-portal-stack-id> <staging-repo-stack-id> <staging-workers-stack-id> <staging-portal-stack-id>

where:
    profile is the name of a profile
    xxx-stack-id is xxx-yy (e.g. '360-0')
